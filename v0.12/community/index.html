Join the HAProxy Ingress community

HAProxy Ingress is an open source project that anyone in the community can use, improve, and enjoy. We'd love you to join us! Here's a few ways to find out what's happening and get involved.

Learn and Connect

Using or want to use HAProxy Ingress? Find out more here:

  • Slack: Chat and get support from other HAProxy Ingress users
  • Users mailing list: Announcements and discussion on a mailing list
  • Stack Overflow: Practical questions and curated answers

Develop and Contribute

If you want to get more involved by contributing to HAProxy Ingress, join us here:

  • GitHub: Development takes place here!

You can find out how to contribute to these docs in our Contribution Guidelines.
